## Getting Started
1. Make sure Node.js and npm are installed.
2. Install yarn.
3. Run `yarn`.

## Commands
1. Development server: `npm run dev` (go to `localhost:8080/public`).
2. Development build: `npm run build:dev`.
3. Production build: `npm run build`.
4. Run tests: `npm t`.
5. Run linter: `npm run lint`.
